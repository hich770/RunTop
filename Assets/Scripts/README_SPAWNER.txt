НАСТРОЙКА ОПТИМИЗИРОВАННОГО СПАВНЕРА ОБЪЕКТОВ

1. НАСТРОЙКА ПУЛА ОБЪЕКТОВ

   - Создайте пустой GameObject и назовите его "ObjectPool"
   - Добавьте на него компонент "ObjectPool"
   - В инспекторе ObjectPool настройте пулы:
     * Для каждого типа объектов добавьте элемент в список Pools
     * Укажите тег (произвольное имя, например "Rock", "Barrel", "Box")
     * Укажите префаб объекта
     * Укажите размер пула (сколько объектов предварительно создать)

2. НАСТРОЙКА ОБЪЕКТОВ

   - На каждый префаб, который вы добавили в пул, добавьте компонент "ObjectReturnToPool"
   - Настройте параметры для этого компонента:
     * Время жизни объекта (lifeTime)
     * Минимальную дистанцию до игрока для отключения (minDistanceToDisable)

3. НАСТРОЙКА СПАВНЕРА

   - На объект с компонентом "Spawner" добавьте список тегов (poolTags)
   - Эти теги должны соответствовать тегам, которые вы указали в ObjectPool
   - Настройте другие параметры спавнера по вашему желанию:
     * Интервал спавна
     * Размер зоны спавна
     * Случайный поворот и масштаб

ДОПОЛНИТЕЛЬНЫЕ СОВЕТЫ:

1. ОПТИМИЗАЦИЯ МАТЕРИАЛОВ
   - Используйте один материал для всех скатывающихся объектов
   - Включите GPU Instancing на материале
   - По возможности используйте атлас текстур

2. ФИЗИКА
   - Установите меньшую частоту обновления физики для далеких объектов
   - Используйте упрощенные коллайдеры (примитивы)

3. ДРУГИЕ ОПТИМИЗАЦИИ
   - Отключите тени для мелких объектов
   - Настройте LOD для сложных моделей

Удачной разработки!
